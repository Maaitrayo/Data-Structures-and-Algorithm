```bash
cd CHAPTER\ (-number-)
```
```bash
cd src
```
```bash
cd executable_files
```
```bash
./program_(-number-)
```

## To compile any c program
```bash
(1) gcc -o (-executable file name-) (-program file name-)
(2) ./(-executable file name-)

example:
(1) gcc -o program_1 program_1.c
(2) ./program_1
```
